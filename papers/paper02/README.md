# Supporting materials for “Do Coding Agents Need Executable World Models, Simplification, and Verification to Solve ARC-AGI-3?”

This directory contains the supporting materials for the article.

**Paper:** [Do Coding Agents Need Executable World Models, Simplification, and Verification to Solve ARC-AGI-3?](http://arxiv.org/abs/2607.15439)

## Contents

- [`agents/`](agents/) contains the runnable agent implementations, prompts, configurations, documentation, and analysis utilities. See [`agents/README.md`](agents/README.md) for an overview.
- [`results/`](results/) contains compact per-game tables for each evaluated agent–model–effort block.

Raw runs are not stored in this repository.

## Released run data

- **Full run artifacts**, including the generated world model for each run, are available in the [Zenodo archive](https://doi.org/10.5281/zenodo.21412274).
- **Final result JSON files only**, without generated artifacts, are available in the [Dropbox archive](https://www.dropbox.com/scl/fi/stp5n7i5j5z75ky3bmz0z/article2_all_runs_jsons.tar.gz?rlkey=7wimse1cz0pdzbf03zze13yjy&st=vrs38j7u&dl=0).
